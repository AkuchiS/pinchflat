pinchflat

Install Pinchflat (the self-hosted YouTube archiver) in a Proxmox LXC via Docker — one command on your host.

View on GitHubSponsor

What it does

One command on your Proxmox host spins up a Debian 12 LXC, installs Docker inside it, and runs Pinchflat — the self-hosted YouTube media manager — via Docker Compose. Pinchflat is distributed only as a Docker image, and this is the clean, supported way to run it on Proxmox. Built for homelabbers.

Full install and usage in the README on GitHub.
